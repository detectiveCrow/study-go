# Go lang

## 왜 go lang 인가
* python 
	* 장점
		* 쉽게 사용가능
	* 단점
		* 느리다
* Java
	* 장점
		* 빠르다
		* 간편하다
	* 단점
		* 복잡도를 증가시킴
* C/C++
	* 장점
		* 빠르다
	* 단점
		* 느린 컴파일 시간
		* 복잡한 타입 시스템

## Go (golang)
* Strong and statically typed
* Excellent community
* Key features
	* simplicity 
		* 간단하면서 배우기 쉬움
	* Fast compile times
		* 빠른 빌드, 빠른 피드백
	* Garbage collected
		* 메모리 관리를 자동화 해줌
	* Build-in concurrency
		* 다중 작업 등
	* Compile to standalone binaries
